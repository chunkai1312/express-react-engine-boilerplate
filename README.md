# express-react-engine-boilerplate

[![JavaScript Style Guide][standardjs-image]][standardjs-url]

> A boilerplate for building express applications using react engine

## Usage

### Setup

```
$ npm install
```

### Run for Development

```
$ npm run dev
```

### Run for Production

```
$ npm start
```
